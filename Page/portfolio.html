Este es mi primer proyecto, lo realicé para un cliente con un emprendimiento de mates elaborados con material biodegradable. Es un modelo simple, pero llamativo y muy User-Friendly. El cliente quedó muy satisfecho.

Este proyecto lo realicé para una empresa de turismo. El sitio Web tiene toda la información del lugar y galerias de imagenes para que la gente pueda apreciar el gran trabajo de la empresa y puedan contactarlos.

Como pueden ver, este sitio Web es un proyecto realizado con la idea de crear un PhotoBlog, un sitio donde subir fotos o experiencias personales y explorar otros perfiles, como si fuera una especie de Facebook.

Este fue un proyecto no muy atractivo a la vista, pero muy útil para el consumidor, ya que es de un chef que quiso subir recetas y tutoriales a un sitio Web y poder ayudar a los principiantes en la cocina.

Este sitio Web es de una empresa de seguro de vida. La idea era la facilitación del acceso a la información y a todo el sistema de guardia, asi como también para consultas médicas y otros servicios de la empresa.

Este proyecto es un sitio para aprender desarrollo web y poder realizar tus propias páginas. La idea fue de un chico emprendedor, que quería brindar cursos gratis y útiles a todo tipo de audiencia.
